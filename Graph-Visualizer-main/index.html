Dijkstra's Algorithm - Shortest Path

Dijkstra's algorithm is used to find the shortest path from a starting node to all other nodes in a weighted graph. It guarantees the shortest path in graphs with non-negative weights.

Steps:

  1. Start with the source node and assign it a tentative distance of 0. All other nodes get Infinity.
  2. Mark all nodes unvisited. Set the current node to the source.
  3. For the current node, check all unvisited neighbors and calculate tentative distances.
  4. Update if the newly calculated distance is smaller than the previously recorded one.
  5. After visiting all neighbors, mark the current node as visited. A visited node will not be checked again.
  6. Select the unvisited node with the smallest tentative distance and repeat until all nodes are visited or the target is reached.
